AI Blog Generator

The AI Blog Generator simplifies creating blog articles from YouTube videos using advanced AI. Built with Django, PostgreSQL, Tailwind CSS, OpenAI, and AssemblyAI, it empowers content creators to convert video content into written articles effortlessly. My role involved designing core features, integrating APIs, creating an intuitive interface, and optimizing performance. Notable achievements include automating transcription and content creation, improving scalability, and receiving positive user feedback.

Spotify Clone

The Spotify Clone project is a Django-based web application that mimics Spotify's music streaming service. It utilizes PostgreSQL for the database and the Spotify API to retrieve music data. Key features include user authentication, music search, and information on top artists and tracks. Users can explore artist profiles, including monthly listeners and top tracks, and stream music. The project uses Bootstrap for design and RESTful APIs to interact with Spotify services. To start, users need to clone the repository, set up the environment, configure the database, and obtain an API key from Spotify.
